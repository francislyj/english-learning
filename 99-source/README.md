# 99-source · 素材库

> 这里是**降级的参考材料**，不是主线。主线在 `00-engine/`。

## 它的身份

《走遍美国》26 集笔记。曾经是学习主线，现在降级为**素材库**：
只在你跑 `00-engine/method.md` 的循环、Step 2 独白卡住、需要某个场景的高频 chunk 时，才翻进来找。

**不要把"读完 26 集"当目标。** 读完它不解决"开不了口"——那是为学日益的陷阱。

## ⚠️ 质量不均，按需取用

- `ep01` 是对照真实台词写的，质量可信。
- 中后段有几集（如 `ep12`）是**凭标题泛写**的（"某个角色被解雇…"），内容不够扎实，别太当真。
- **修订原则：哪一集你真翻进去、发现不够用，那一集再补**——从卡住的地方往里长，不预先把整个库填满。
