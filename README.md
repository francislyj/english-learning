# english-learning

> 这不是一个「读物」仓库，是一台**每日输出引擎**。
> 目标只有一个：把被动词汇逼成主动，让我每天**开口**，而不是每天多读。

## 怎么用（每天 15 分钟）

打开 `00-engine/method.md`，跑那个循环。就这一件事。

> 限时回忆 → 真实场景独白 + 录音 → 标记缺口。明天再来。

## 目录

| 目录 | 装什么 | 什么时候看 |
| --- | --- | --- |
| `00-engine/` | **主线**：方法论 + 主动调取追踪表 | 每天 |
| `01-daily/` | 每日输出日志（一天一篇） | 每天写一篇 |
| `02-scenarios/` | 我的真实开口场景（PR / 大会 / 客户 / 交易）的高频 chunk + 独白题 | 选独白题、查 chunk 时 |
| `99-source/` | 素材库：《走遍美国》26 集笔记（已降级，需要时才翻） | 偶尔补 chunk |
| `99-assets/` | 录音、图片 | 存录音 |

## 两条铁律

1. **每天 reps > 每周进度** —— 哪怕 5 分钟，也跑一遍限时回忆，别断链。
2. **录音不可省** —— 不录音 = 没碰到「不敢开口」那层恐惧。想跳过录音的念头，正是要练的东西。

## 在 Obsidian 中打开

`Open folder as vault` → 选本目录。
