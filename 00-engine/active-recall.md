# 主动调取追踪表（Active Recall）

> 这张表是引擎的燃料。每天 Step 1 只练这里**没毕业**的 chunk。
> 一个 chunk 的目标不是「看懂」，是「中文一出现，英文 0.5 秒脱口而出」。

## 怎么用

1. 独白时「想说但说不出」的表达 → 加到这里，状态记 🔴。
2. 每天 Step 1：**只看中文列，盖住英文**，大声说。
3. 说出来了 → 升一级；卡住 → 保持/降级 🔴，明天优先。
4. 连续 3 天都 🟢 自动 → 标 ✅ 毕业，移到底部存档，不再每天练。

## 状态图例

| 状态     | 含义             | 处理     |
| ------ | -------------- | ------ |
| 🔴 被动  | 看懂，但开口说不出 / 卡住 | 每天优先练  |
| 🟡 能想起 | 想几秒能说出来        | 继续练，提速 |
| 🟢 自动  | 脱口而出，不用想       | 巩固中    |
| ✅ 毕业   | 连续 3 天 🟢      | 存档，停练  |
