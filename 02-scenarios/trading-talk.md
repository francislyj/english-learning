# 场景：市场 / 交易交流

> 用英语聊行情、复盘、风险。也是你另一个道场的「开口」延伸。
> 注意：很多交易表达是固定行话（chunk 化程度极高），背下来直接用。

## 描述行情 / 判断

| 中文 | English chunk |
| --- | --- |
| 我看多/看空 | I'm bullish / bearish on it. |
| 它在 ___ 附近有支撑 | It's finding support around ___. |
| 这波涨势看起来要竭了 | This rally looks like it's running out of steam. |
| 现在是震荡市 | It's range-bound right now. |
| 我在等一个回调 | I'm waiting for a pullback. |

## 仓位 / 操作

| 中文 | English chunk |
| --- | --- |
| 我建了点仓 | I took a small position. |
| 我止盈了一半 | I took half off the table. |
| 我被止损了 | I got stopped out. |
| 我空仓观望 | I'm sitting on the sidelines. |

## 承认看错（执着的解药——能松口说这句，本身就是修行）

| 中文 | English chunk |
| --- | --- |
| 我这次看错了 | I got this one wrong. |
| 我的判断是错的 | My read was off. |
| 我太早进场了 | I jumped in too early. |
| 我应该早点止损 | I should've cut it sooner. |

## 聊风险 / 心态

| 中文 | English chunk |
| --- | --- |
| 我只赌我输得起的 | I only risk what I can afford to lose. |
| 我在控制仓位 | I'm keeping my size small. |
| 别和市场较劲 | Don't fight the market. |
| 现金也是一种仓位 | Cash is a position too. |

---

## 独白题目（录音用）

1. 60 秒：用英语复盘你最近的一笔交易——为什么进、结果如何、学到什么。
2. 45 秒：描述你现在对某个市场的判断 + 你的理由。
3. 30 秒：讲一笔你看错的交易，承认错在哪（练「松口认错」这件事）。
