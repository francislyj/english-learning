# 场景：客户 / 合作 call

> 这是最高压的场景——涉及钱、拒绝、争取认可，恐惧的指纹最容易冒出来。
> 越是这里，越要靠**背熟的 chunk**顶住，别现场组装。

## 开场 / 寒暄

| 中文 | English chunk |
| --- | --- |
| 谢谢抽时间 | Thanks for taking the time. |
| 听得清吗？ | Can you hear me okay? |
| 那我们开始吧？ | Shall we dive in? |

## 报价 / 谈条件

| 中文 | English chunk |
| --- | --- |
| 这个项目我的报价是 ___ | For this project, my rate would be ___. |
| 我按 ___ 收费 | I charge ___ per ___. |
| 这超出范围了，得另算 | That's outside the scope, so it'd be extra. |
| 我们可以分阶段做 | We could do this in phases. |

## 拖时间 / 不当场答应（关键——别因为怕冷场就乱承诺）

| 中文 | English chunk |
| --- | --- |
| 我需要想想再答复你 | Let me think about it and get back to you. |
| 这个我得查一下再确认 | Let me check on that and confirm. |
| 我今天结束前给你答复 | I'll get back to you by end of day. |

## 礼貌拒绝（恐惧最爱让你降价/接烂活——这里顶住）

| 中文 | English chunk |
| --- | --- |
| 这个我恐怕接不了 | I'm afraid I can't take that on. |
| 这不太是我的方向 | That's not quite my area. |
| 现阶段我精力有限 | I'm at capacity right now. |
| 这个预算我做不了 | I can't make that work at this budget. |

## 争取认可 / 收尾

| 中文 | English chunk |
| --- | --- |
| 我很有信心能交付 | I'm confident I can deliver this. |
| 我之前做过类似的 | I've done something similar before. |
| 期待合作 | Looking forward to working together. |

---

## 独白题目（录音用）

1. 60 秒：客户问你这个项目多少钱，你报价 + 说清包含什么、不包含什么。
2. 45 秒：客户想压价 / 加需求不加钱，你礼貌地守住底线。
3. 30 秒：客户当场要你答应一件你没把握的事，你不拒绝也不乱答应，先要时间。
