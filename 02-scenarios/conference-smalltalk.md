# 场景：技术大会 / 开发者社交破冰

> 你最常卡的不是语法，是「开场第一句」和「介绍自己在做什么」。
> 下面是高频弹药。卡住时翻这里；练熟的搬进 `00-engine/active-recall.md`。

## 破冰开场

| 中文 | English chunk | 情绪 |
| --- | --- | --- |
| 嗨，这个 talk 怎么样？ | Hey, what'd you think of that talk? | 😊 轻松 |
| 你也是做后端的吗？ | Are you on the backend side too? | 🤔 试探 |
| 我之前没在这儿见过你 | I don't think we've met. | 👋 主动 |
| 这地方人真多啊 | Quite a turnout, huh? | 😄 闲聊 |

## 介绍自己（最该练熟的）

| 中文 | English chunk |
| --- | --- |
| 我是个独立开发者 | I'm an indie developer. / I build my own apps. |
| 我在做一个自己的 App | I'm building an app of my own. |
| 它帮人 ___（一句话产品） | It helps people ___. |
| 我主要写 Go，最近在碰 Swift | I mostly write Go, dabbling in Swift lately. |
| 我自己一个人全栈做 | I'm a one-person full-stack shop. |

> ⚠️ 「一句话说清你的 App」是你反复卡的点。**先写死一个 10 秒版本背下来**，别每次现编。

## 接话 / 夸对方

| 中文 | English chunk |
| --- | --- |
| 那听起来挺酷的 | That sounds really cool. |
| 你们怎么解决 ___ 的？ | How'd you guys handle ___? |
| 我一直想试试那个 | I've been meaning to try that. |
| 哇，这个我得了解一下 | Oh nice, I'll have to check that out. |

## 收尾 / 交换联系方式

| 中文 | English chunk |
| --- | --- |
| 很高兴认识你 | Great meeting you. |
| 我们 GitHub 上联系？ | Let's connect on GitHub? |
| 我加你？ | Can I add you? |

---

## 独白题目（录音用）

1. 60 秒：在大会上跟一个陌生开发者破冰，介绍你在做什么 App，反问对方在忙什么。
2. 30 秒：用一句话 + 展开两句，说清你的产品解决什么问题。
3. 45 秒：你听完一个开源作者的 talk，上前夸他 + 问一个技术问题。
