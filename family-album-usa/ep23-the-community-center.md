# Episode 23: The Community Center

## 场景概述

社区中心面临问题（可能是资金或关闭风险），居民们组织起来采取行动。涉及公共事务讨论、提出方案、团队合作。

## 核心句型

| 句型 | 用途 | 例句 |
|------|------|------|
| We need to do something about this. | 我们得为此做点什么 | — 号召行动 |
| I'd like to propose... | 我想提议… | — 正式提出方案 |
| What do you suggest? | 你建议怎么做？ | — 征求方案 |
| Let's put it to a vote. | 我们投票决定吧 | — 民主决策 |
| I agree with... | 我同意… | — 表态支持 |
| We can make it work. | 我们能让它成功 | — 表达信心 |

## 重点词汇

| 词/短语 | 释义 | 语境 |
|---------|------|------|
| community center | 社区活动中心 | 本集场景核心 |
| fundraiser | 筹款活动 | organize a fundraiser |
| petition | 请愿书 | sign a petition |
| proposal | 提案 | submit a proposal |
| committee | 委员会 | form a committee |
| participate | 参与 | participate in the meeting |
| unanimous | 一致同意的 | a unanimous decision |

## 语法点

1. **We need to + 动词** — We need to raise money. / We need to act now.
2. **I'd like to + 动词 (正式提议)** — I'd like to suggest an alternative.
3. **Let's + 动词 (集体决定)** — Let's form a committee. / Let's write a letter.

## 文化注释

- **Community center** — 美国社区的公共活动场所，提供课程、运动、聚会空间
- **Town hall meeting** — 居民讨论公共事务的传统形式，人人可发言
- **Grassroots activism** — 美国人重视"自下而上"解决社区问题
